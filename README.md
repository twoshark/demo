# C# in Docker Example

# Running Locally

```
dotnet run .
```

# Docker Build

```
docker build . --tag twoshark/docker-cs-demo --no-cache --progress plain
```

# Run with Docker Compose
```
docker-compose up
```

# Accessing the site
- [Running locally or with docker](http://localhost:8080)
- [Running with docker-compose](http://localhost:8000)
